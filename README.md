# 財務省になろう！ 〜 補佐官ミライと描く日本の明日 〜

## 概要

これは、日本のマクロ経済をシミュレーションするゲームです。プレイヤーは財務次官となり、税率、政府支出、金利といった政策を駆使して、国民の支持を維持しながら経済を成長させ、デフレを克服することを目指します。

ゲームは、実際の日本の経済データ（1994年〜2023年）に基づいた計量経済学モデル（誤差修正モデルなど）を元に動作しています。

## 主な使用技術

このアプリケーションは、以下の技術で構築されています。

*   **HTML5**
*   **CSS3**
    *   [Tailwind CSS](https://tailwindcss.com/): UIのスタイリングに使用しています。
*   **JavaScript (ES6+)**
*   **UIライブラリ**
    *   [React](https://react.dev/): ゲーム画面のUI構築に利用しています（CDN経由）。
    *   [Recharts](https://recharts.org/): 経済指標のグラフ表示に利用しています（CDN経由）。
    *   [Lucide Icons](https://lucide.dev/): アイコン表示に利用しています（CDN経由）。
    *   [KaTeX](https://katex.org/): 開発コラムページの数式表示に利用しています（CDN経由）。

## 実行方法

このアプリケーションはサーバーを必要としないため、ローカル環境で簡単に実行できます。

1.  このリポジトリのファイルをダウンロードします。
2.  `index.html` ファイルをウェブブラウザで開いてください。

以上でゲームを開始できます。

**（推奨）ローカルサーバーでの実行**
もしファイルの読み込み等で問題が発生する場合（特にファイルを変更する場合）は、以下の方法でローカルサーバーを起動することをお勧めします。

お使いのPCにPythonがインストールされている場合：
1.  ターミナル（コマンドプロンプト）を開き、このプロジェクトのディレクトリに移動します。
2.  以下のコマンドを実行します。
    ```bash
    # Python 3.x の場合
    python -m http.server
    ```
3.  ブラウザで `http://localhost:8000` にアクセスします。
